Misión Primaria

Nuestro objetivo principal es recibir datos atmosféricos durante el descenso del CanSat:

Sensor BMP280

Utilizaremos un sensor BMP280 (o el AdaFruit MPL3115A2) para medir:

  • Presión atmosférica
  • Temperatura ambiente

Los datos se transmitirán mediante ondas de radio (módulo APC220) e incluirán:

  • Número de paquete en la secuencia
  • Nombre del equipo
  • Valores separados por comas

Diseño del Paracaídas

Paracaídas esférico de tela Ripstop con:

  • Radio de 12.66 cm (calculado con fórmulas aerodinámicas)
  • 8 gajos diseñados matemáticamente
  • Sistema anti-G (quitavueltas)
  • Velocidad de descenso objetivo: 9.5 m/s

Fórmulas Utilizadas

Para el cálculo del área del paracaídas:

mg = ½CDρAv2

Donde:

  • m: Masa del CanSat
  • g: Aceleración gravitatoria
  • CD: Coeficiente de arrastre
  • ρ: Densidad del aire
  • A: Área del paracaídas
  • v: Velocidad de descenso

Diseño de la Carcasa

Nuevo modelo diseñado en Sketch con:

  • 4 orificios para unión con tornillos
  • Sistema de ventilación mejorado
  • Baldas ajustables a diferentes alturas
  • Cumple con las dimensiones reglamentarias (66mm diámetro × 115mm altura)

Misión Secundaria

Sistema avanzado de seguimiento y predicción de trayectoria:

Seguimiento en Tiempo Real

Implementamos un módulo GPS NEO6MV2 que:

  • Proporciona coordenadas exactas
  • Se visualiza en Google Maps mediante software Python
  • Almacena datos en una base de datos local

Predicción de Trayectoria

Desarrollamos algoritmos que calculan:

  • Punto de aterrizaje estimado
  • Velocidad instantánea (comparando puntos cada 1 segundo)
  • Altura inicial (del sensor BMP280)

Ecuaciones principales:

x = x0 + v0xt + ½gt2

y = y0 + v0yt + ½gt2

Comunicación por Radio

Usamos el módulo APC220 para:

  • Transmitir datos de sensores
  • Recibir coordenadas GPS
  • Comunicación bidireccional con la estación base
